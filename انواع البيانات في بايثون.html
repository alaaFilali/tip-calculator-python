انواع البيانات في بايثون

رأينا في الدروس السابقة اننا لحساب عدد الحروف في string ما نستعمل الدالة len 


print(len("Alaa"))
            

4

الان ماذا سوف يحدث لو مررت داخل اقواس الدالة عدد وليس string ؟ 


print(len(12345))

سوف احصل على خطأ من نوع TypeError

سابقا راينا نوع بيانات وهو سلاسل الحروف string ولكن هناك انواع اخرى وسوف نعرف بعضها في درس اليوم كالاعداد الصحيحة والاعداد بفاصلة و boolean

ال string هي سلسلة من الحروف ونعرفها من خلال علامات التنصيص في البداية والنهاية 


"Hello"

فلو اردت طباعة الحرف الاول من ال string استعمل علامات [] ثم داخلها اضع رقم الحرف في الكلمة و الحرف الاول عند 0 


print("Hello"[0])
            

H

لطباعة الحرف الاخير 


print("Hello"[4])
            

o

لو وضعت عددا داخل علامات تنصيص فسوف يعتبر string وليس عددا 


"123"

وسوف يعامل معاملة ال string وليس كالاعداد العادية 


print("123" + "456")
            

123456

اما اذا اردت عملية الجمع العادية بين عددين فقط قم بكتابة العدد بدون اي شيء وبدون علامات التنصيص 


print(123 + 456)
            

579

يمكنك اضافة علامة _ بين الاعداد وسوف يتم تجاهلها 


print(123_456_789)
            

123456789

هذه الاعداد بدون فاصلة تسمى integer والتي بالفاصلة تسمى float 


3.14

والفاصلة هي النقطة وليس الفاصلة العادية

والنوع الاخير من انواع البيانات هو ال boolean وهو ابسط نوع لان لديه فقط قيمتين True او False 


True
