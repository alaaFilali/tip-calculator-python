تحويل انواع البيانات

اليك الكود الموالي: 


num_char = len(input("What's your name?: "))
print("Your name has " + num_char + " characters.")

عند تشغيل البرنامج سوف يسألك ان تدخل اسمك ثم يخزن في متغير num_char ثم يقوم بطباعة جملة اسمك فيه عدد معين من الحروف ولكن ماسوف تحصل عليه هو error من نوع TypeError والسبب هو انك تريد عمل concatination اي لصق بين الجملة الاولى مع قيمة المتغير num_char ثم مع الجملة الاخرى وهذا غير ممكن لان عملية اللصق تحدث بين strings اما نوع المتغير num_char ليس string لانه يرجع عدد الاحرف في string معين اي ان نوعه هو int اي عدد بدون فاصلة

اذن كيف يمكننا تجنب هذا النوع من الاخطاء؟ كيف يمكننا معرفة نوع المتغير؟ الاجابة يمكننا استعمال دالة type وسوف ترجع لك نوع ما اعطيته لها 


num_char = len(input("What's your name?: "))
print(type(num_char))
            

What's your name?: Alaa
< class 'int' >

قال لي ان نوع المتغير num_char هو int

الان لنجعل عبارة الطباعة السابقة تعمل لابد ان نحول المتغير num_char الى string ويمكننا فعل ذلك عن طريق دالة str 


num_char = len(input("What's your name?: "))
new_num_char = str(num_char)
print("Your name has " + new_num_char + " characters.")
            

What's your name?: Alaa 
Your name has 4 characters.

انت لست محدود بتحويل فقط الاعداد الى strings يمكنك تحويل مختلف الانواع وهذه بعض الامثلة: 


a = 123
print(type(a))

a = str(123)
print(type(a))

a = float(123)
print(type(a))
            

< class 'int' >
< class 'str' >
< class 'float' >

الان ماذا تعتقد سوف تكون نتيجة هذا السطر 


print(70 + float("100.5"))

الاجابة: 


170.5

وما حدث خلف الكواليس هو تم تحويل "100.5" الذي هو string الى float ثم تمت عملية الجمع بينه وبين العدد 70 والنتيجة هي العدد 170.5 من نوع float لان به فاصلة

الان ماذا عن نتيجة هذا السطر: 


print(str(70) + str(100))
            

70100

كملخص للدرس يمكنك استعمال الدالة type لمعرفة نوع المتغير الذي تتعامل معه ويمكنك استعمال الدالة str او int او float لتحول الى النوع الذي تريده.
